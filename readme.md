## 开坑写一个校园社区网站

主要功能目前大概有:
- 手机端API
- 校园众包功能
- 校园表白墙功能
- 活动众筹功能
- 校园文学
- 校园话题与交流

Laravel使用版本:5.4,后续会根据Laravel版本的升级来升级使用的版本
